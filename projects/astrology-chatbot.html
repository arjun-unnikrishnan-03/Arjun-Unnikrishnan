ACADEMIC AI PROJECT

Astrology Prediction Chatbot

AI-powered astrology chatbot using NLP, Machine Learning, and Deep Learning

Python Flask NLP PyTorch Scikit-learn Decision Tree Neural Networks

THE PROBLEM

Traditional astrology platforms struggle with real-time personalization, slow manual interpretation, and generic horoscope generation. Users expect instant, accurate, and conversational responses based on their birth details and astrological context.

SYSTEM OVERVIEW

This project implements an AI-powered astrology chatbot that combines Natural Language Processing (NLP), Machine Learning, and Deep Learning to deliver personalized astrology predictions. The system processes user queries in real time, classifies intent using a neural network, and generates astrology-based responses using trained models.

KEY FEATURES

  • 01. NLP-Based Intent Classification using Neural Networks (PyTorch)
  • 02. Decision Tree Model for Personalized Astrology Recommendations
  • 03. Real-Time Chatbot Interface with Flask REST API
  • 04. Horoscope, Career, Compatibility & Zodiac-Based Predictions
  • 05. User Input Validation & Confidence-Based Response Filtering
  • 06. Modular Architecture (NLP Engine, ML Models, Web Interface)

THE TECHNICAL DEPTH

The chatbot uses tokenization, stemming, and bag-of-words representation to preprocess user input. A custom neural network trained with CrossEntropyLoss and Adam optimizer classifies user intent. Predictions are served through Flask APIs and consumed by a real-time web UI.

THE OUTCOME

This project demonstrates my ability to design and implement AI-driven conversational systems using NLP and ML. It highlights hands-on experience with model training, inference pipelines, backend API integration, and building end-to-end intelligent applications.

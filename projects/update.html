FEATURED PROJECT

GroceEazy Organic

React 19 TypeScript Redux Toolkit Node.js MongoDB Redis Docker

THE PROBLEM

Most grocery platforms fail at scalability, performance, and maintainability. GrocEazy was built to solve real-world challenges like high-volume product access, secure authentication, admin control, and fast search — all within a clean, modular architecture.

KEY FEATURES

  • 01. Decoupled Client-Server Architecture (SPA + REST API)
  • 02. Secure Authentication with JWT, Argon2 & Google OAuth
  • 03. Role-Based Admin System for Product & Order Management
  • 04. Redis Caching Layer for High-Performance Product Retrieval
  • 05. Image Upload & CDN Optimization using Cloudinary
  • 06. AI-Powered Recipe Suggestions (AI Chef Concept)
  • 07. Type-Safe Validation using Zod, Joi & Express-Validator

THE RESULT

GrocEazy demonstrates my ability to design and build production-ready full-stack systems — focusing on scalability, security, performance optimization, and clean code structure. The project includes extensive documentation, Dockerized deployment, and a real-world admin-customer workflow.

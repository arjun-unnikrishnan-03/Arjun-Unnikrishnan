01 — Problem

THE PROBLEM

Most grocery platforms fail at scale, transactional integrity, and operator UX. GrocEazy was built to solve real-world problems: high-volume catalog access, race-free inventory decrement, secure multi-role authentication, real-time customer support, and an AI-assisted shopping experience — all in a clean, modular monorepo.

02 — What I Built

IMPACT BULLETS

  • Architected a TypeScript monorepo (Express 5 + Mongoose 9 backend, React 19 + Vite + Redux Toolkit frontend) implementing 9 RESTful resource domains under a versioned /api/v1 router with role-based route guards (customer / manager / admin).
  • Designed a hardened JWT auth system using Argon2-hashed passwords + composite refresh tokens (sessionId:rawToken for O(1) lookup), automatic token rotation with replay-detection and session revocation, OTP email verification, password reset, and Google OAuth — with rate limiting on every auth endpoint and a silent-refresh Axios interceptor.
  • Implemented transactional order processing with MongoDB session.startTransaction() to atomically validate stock, decrement inventory via conditional findOneAndUpdate({ stock: { $gte: qty } }), build line items, clear cart, and trigger low-stock email alerts — with symmetric stock restoration on cancellation.
  • Built a real-time customer support module with Socket.IO rooms, persisted ChatMessage history, typing indicators, unread-count broadcasts, and a least-loaded-manager auto-assignment algorithm using $lookup + $size aggregation.
  • Engineered the customer storefront with React Router 7 protected routes, redux-persist state, Cloudinary uploads, faceted product search (text + price + dietary + stock filters with regex-escaped multi-word matching), Leaflet delivery maps, html2canvas/jsPDF invoice download, Web Speech API voice search, and a Gemini-powered AI Chef.
  • Hardened the platform with Helmet, CORS, Zod + Joi validation, Winston structured logging, compound + text indexes for product search, soft-delete patterns, $facet pagination, a multi-stage Node 22 Alpine Dockerfile, ESLint + Prettier + Husky + commitlint pre-commit gates, and Vercel deployment.
03 — Result

THE RESULT

GrocEazy demonstrates the full surface area of production-ready full-stack engineering — from race-free transaction design and real-time messaging to operator UX, AI integration, and DevOps. It's the project I point to when asked: "have you shipped something real?"
